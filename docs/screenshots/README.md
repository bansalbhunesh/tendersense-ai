# Screenshots for judges / README

Add PNGs here and link them from the root [`README.md`](../../README.md) **Demo media** section. Recommended width: **1200–1600 px**. Keep filenames stable so README links do not break.

## Required captures (minimum)

| File | What to show |
|------|----------------|
| `dashboard.png` | Officer dashboard with at least one tender card and pagination or stats visible. |
| `reasoning-graph.png` | Workspace reasoning graph with a verdict-colored node and side panel or evidence chip. |
| `review-queue.png` | Two-pane review queue with criterion row and override or status visible. |

## Recording (video)

1. **60–90 seconds:** register → upload demo PDF from `demo/pdfs/` → add two bidders → run evaluate → open reasoning graph → show one override or review action.
2. Export **1080p** if possible; host on Loom / YouTube (unlisted) and paste the URL in README **Demo media**.

## Tips

- Use a **demo officer** account; blur any real agency emails if needed.
- Light mode is fine; ensure contrast is readable when scaled down in README.
